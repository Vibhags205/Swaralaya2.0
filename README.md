Swaralaya 2.0 is a full-stack web application designed for Indian classical music learners and enthusiasts.  
It provides learning resources, raga exploration, instrument audio samples, and an AI-powered music assistant.

The platform supports both **English and Kannada** languages.

---

##  Features:

-  Bilingual Support (English & Kannada)
-  Featured Blog Posts
-  Raga Explorer with Audio Playback
-  Instrument Gallery with Audio Samples
-  AI Chatbot for Music Queries
-  Email Subscription System

---

##  Tech Stack:

### Frontend
- React.js (Vite)
- React Router
- Firebase Hosting
- CSS (Custom Styling)

### Backend
- Node.js
- Express.js
- Groq API
- Firebase Admin SDK

### Database & Services:
- Firebase (Subscriptions & Admin)
- Render (Backend Hosting)
